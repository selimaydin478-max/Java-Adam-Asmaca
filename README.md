# Java-Adam-Asmaca
# Dinamik Adam Asmaca Oyunu (Java Swing)

Bu proje, Programlama Dilleri II dersi kapsamında geliştirilmiş, dosya işlemleri (I/O) ve dinamik arayüz yönetimi içeren bir masaüstü kelime tahmin oyunudur.

## 🚀 Proje Özellikleri
* **Dinamik Veri Çekme:** Oyundaki kelimeler, şifreler ve adam asmaca görselleri koda gömülü değildir; dışarıdaki klasörlerden anlık olarak okunur.
* **Gelişmiş Arayüz:** Java Swing (JTabbedPane, JPanel, JTable) kullanılarak çok sekmeli bir yapı tasarlanmıştır.
* **Skor ve Log Yönetimi:** Oynanan oyunların sonuçları ve süreleri `.txt` dosyalarına kaydedilir ve tablolar halinde listelenir.
* **Güvenlik:** Log ve skor temizleme işlemleri, dışarıdan güncellenebilen şifre korumalı bir sistemle yetkilendirilmiştir.

## 🛠️ Kullanılan Teknolojiler
* Java (JDK)
* Java Swing (GUI)
* Dosya İşlemleri (File I/O)
* Apache NetBeans IDE
